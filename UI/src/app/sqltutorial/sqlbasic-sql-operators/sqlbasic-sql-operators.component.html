SQL Operators

Operators are used to perform operations on data stored in a database.

Types of SQL Operators

  • Arithmetic Operators (+, -, *, /)
  • Comparison Operators (=, !=, <, >, <=, >=)
  • Logical Operators (AND, OR, NOT)

Arithmetic Operators

Example (Calculating New Balance after Interest in Banking):

SELECT AccountID, Balance * 1.05 AS NewBalance FROM Accounts;

Comparison Operators

Example (Finding Customers with High Balance):

SELECT * FROM Accounts WHERE Balance > 50000;

Logical Operators

Example (Finding VIP Customers with High Balance and Savings Account):

SELECT * FROM Accounts WHERE Balance > 50000 AND AccountType = 'Savings';
