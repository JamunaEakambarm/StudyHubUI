SQL CRUD Operations

CRUD stands for Create, Read, Update, and Delete ‚Äî the four fundamental operations used to interact with data in relational databases.

While "CREATE TABLE" is essential to define the structure of your database, it's not part of the CRUD operations. CRUD specifically refers to the manipulation of data within an existing table. It focuses on inserting data (Create), retrieving data (Read), modifying data (Update), and removing data (Delete). These actions are performed after the table structure is already established.

Data Records

CREATE: Insert new records into the database, e.g.:

INSERT INTO Customers (CustomerID, FirstName, LastName, Email, DateOfBirth)
  VALUES (1, 'Alice', 'Johnson', 'alice@email.com', '1990-05-10');

READ: Fetch data from the database, e.g.:

SELECT * FROM Customers WHERE Email = 'alice@email.com';

UPDATE: Modify existing records, e.g.:

UPDATE Customers SET Email = 'alice.new@email.com' WHERE CustomerID = 1;

DELETE: Remove records from the database, e.g.:

DELETE FROM Customers WHERE CustomerID = 1;

Summary

CRUD Operations:

  • Create: Insert data into a table.
  • Read: Retrieve data from a table (using SELECT).
