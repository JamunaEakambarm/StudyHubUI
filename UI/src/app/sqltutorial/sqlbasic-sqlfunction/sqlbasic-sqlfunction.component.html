SQL Functions

What are SQL Functions?

Functions perform calculations on data and return results.

Types of Functions

  • Aggregate Functions (SUM, AVG, COUNT, MAX, MIN)
  • String Functions (UPPER, LOWER, LENGTH)
  • Date Functions (NOW, YEAR, MONTH)

Aggregate Functions

Example (Calculating Total Deposits in the Bank):

SELECT SUM(Balance) AS TotalDeposits FROM Accounts;

String Functions

Example (Converting Customer Name to Uppercase):

SELECT UPPER(FirstName) FROM Customers;

Date Functions

Example (Calculating Customer Age from Date of Birth):

SELECT FirstName, YEAR(CURDATE()) - YEAR(DateOfBirth) AS Age FROM Customers;
