Subqueries in SQL

What is a Subquery?

A subquery is a query nested inside another query to retrieve data based on a condition.

Types of Subqueries:

  • Single-row subqueries – Return one value.
  • Multi-row subqueries – Return multiple values.
  • Correlated subqueries – Execute once for each row.

Example (Finding Customers with High Balance Accounts)

SELECT FirstName, LastName FROM Customers WHERE CustomerID IN (SELECT CustomerID FROM Accounts WHERE Balance > 100000 );

🔹 How it works: The subquery finds CustomerIDs where balance > 100,000, and the main query retrieves customer details.
