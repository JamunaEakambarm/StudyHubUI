Full-Text Search in SQL

What is Full-Text Search?

Full-Text Search (FTS) is a technique used in SQL databases to search for text efficiently. It enables complex searching capabilities beyond the LIKE operator.

  • Faster than LIKE for large text fields.
  • Supports advanced searches like boolean and phrase search.
  • Helps in searching natural language queries.

Types of Full-Text Search

Type Description
Basic Search Searches for single or multiple words.
Phrase Search Finds exact phrases in text.
Boolean Search Uses logical operators (AND, OR, NOT) for complex queries.
Proximity Search Finds words appearing close to each other.
Weighted Ranking Ranks search results based on relevance.

Basic Full-Text Search

Creating a Full-Text Index

CREATE FULLTEXT INDEX ON Products(ProductDescription);

Single Word Search

SELECT * FROM Products WHERE CONTAINS(ProductDescription, 'laptop');

Multiple Word Search

SELECT * FROM Products WHERE CONTAINS(ProductDescription, 'laptop OR tablet');

Phrase Search

Searching for an Exact Phrase

SELECT * FROM Products WHERE CONTAINS(ProductDescription, '"gaming laptop"');

Boolean Search

Using Boolean Operators

SELECT * FROM Products WHERE CONTAINS(ProductDescription, '"gaming" AND "laptop"');
SELECT * FROM Products WHERE CONTAINS(ProductDescription, '"laptop" AND NOT "used"');

Proximity Search

Finding Words Near Each Other

SELECT * FROM Products WHERE CONTAINS(ProductDescription, 'NEAR((gaming, laptop), 3)');

Weighted Ranking

Ranking Search Results

SELECT ProductName,  
         RANK() OVER (ORDER BY KEY_TBL.RANK DESC) AS Rank  
  FROM Products  
  JOIN CONTAINSTABLE(Products, ProductDescription, 'laptop') AS KEY_TBL  
  ON Products.ProductID = KEY_TBL.[KEY];
