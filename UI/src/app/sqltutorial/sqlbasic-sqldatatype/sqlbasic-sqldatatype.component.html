SQL Server Data Types

Exact Numerics

Data Type Description Example
tinyint Stores integers from 0 to 255. CREATE TABLE Employees (Age tinyint);
smallint Stores integers from -32,768 to 32,767. CREATE TABLE Employees (EmployeeID smallint);
int Stores integers from -2,147,483,648 to 2,147,483,647. CREATE TABLE Employees (EmployeeID int);
bigint Stores very large integers. CREATE TABLE Transactions (TransactionAmount bigint);
bit Stores Boolean values (0, 1, NULL). CREATE TABLE Employees (IsActive bit);
decimal/numeric Stores exact numeric values with precision and scale. CREATE TABLE Products (Price decimal(10, 2));
money Stores monetary values. CREATE TABLE Products (Price money);
smallmoney Stores smaller monetary values. CREATE TABLE Transactions (Amount smallmoney);

Approximate Numerics

Data Type Description Example
float Stores approximate numeric values with floating decimal points. CREATE TABLE Measurements (Value float);
real Similar to float but with less precision. CREATE TABLE Temperatures (Temp real);

Date and Time

Data Type Description Example
date Stores only the date part. CREATE TABLE Events (EventDate date);
time Stores only the time part. CREATE TABLE WorkHours (StartTime time);
datetime2 Stores date and time with greater precision. CREATE TABLE Appointments (Appointment datetime2);
datetimeoffset Stores date and time with time zone offset. CREATE TABLE Meetings (MeetingTime datetimeoffset);
datetime Stores both date and time, with less precision. CREATE TABLE Logs (Timestamp datetime);
smalldatetime Stores date and time with less precision. CREATE TABLE Events (EventTime smalldatetime);

Character Strings

Data Type Description Example
char Fixed-length character data. CREATE TABLE Users (Username char(10));
varchar Variable-length character data. CREATE TABLE Users (Description varchar(255));
text Variable-length character data, deprecated. CREATE TABLE Documents (Content text);

Unicode Character Strings

Data Type Description Example
nchar Fixed-length Unicode data. CREATE TABLE Users (Name nchar(100));
nvarchar Variable-length Unicode data. CREATE TABLE Products (Name nvarchar(200));
ntext Variable-length Unicode text data, deprecated. CREATE TABLE Notes (Text ntext);

Binary Strings

Data Type Description Example
binary Fixed-length binary data. CREATE TABLE Files (FileHash binary(16));
varbinary Variable-length binary data. CREATE TABLE Files (Data varbinary(max));
image Stores large binary data, deprecated. CREATE TABLE Images (Photo image);

Other Data Types

Data Type Description Example
cursor Used to hold a reference to a cursor. DECLARE MyCursor CURSOR FOR SELECT * FROM Employees;
geography Stores spatial data (geographic coordinates). CREATE TABLE Locations (GeoLocation geography);
geometry Stores spatial data (geometric shapes). CREATE TABLE Shapes (Shape geometry);
hierarchyid Stores hierarchical data. CREATE TABLE Organization (Manager hierarchyid);
json Stores JSON-formatted data. CREATE TABLE Orders (OrderData json);
vector Stores vectors for machine learning. CREATE TABLE MLData (Feature vector);
rowversion Used for versioning rows in a table. CREATE TABLE Employees (RowVersion rowversion);
sql_variant Stores values of any SQL Server data type. CREATE TABLE Products (ProductData sql_variant);
table Stores a result set in a variable. DECLARE @ProductTable table (ProductID int, Name varchar(100));
uniqueidentifier Stores GUID (Globally Unique Identifier). CREATE TABLE Orders (OrderID uniqueidentifier);
xml Stores XML-formatted data. CREATE TABLE Products (ProductInfo xml);
