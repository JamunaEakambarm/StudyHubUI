📌 Introduction to SQL & Relational Databases

SQL (Structured Query Language) is used to interact with **relational databases**. It helps store, retrieve, update, and delete data efficiently.

📂 What is a Relational Database?

A **relational database** stores data in **tables** and maintains **relationships** between them.

🎓 Example: School Database

📄 Students Table

student_id name age city
1 Alice 14 New York
2 Bob 15 Chicago

📄 Courses Table

course_id course_name teacher
101 Math Mr. Smith
102 Science Mrs. Brown

🛠️ Basic SQL Commands

🔹 Create a Table

CREATE TABLE Students (
   student_id INT PRIMARY KEY,
   name VARCHAR(50),
   age INT,
   city VARCHAR(50)
);

🔹 Insert Data

INSERT INTO Students (student_id, name, age, city) VALUES
   (1, 'Alice', 14, 'New York'),
   (2, 'Bob', 15, 'Chicago');

🔹 Retrieve Data

SELECT * FROM Students;

🔹 Update Data

UPDATE Students SET age = 16 WHERE student_id = 2;

🔹 Delete Data

DELETE FROM Students WHERE student_id = 1;

🎯 Practice these SQL queries in a real database system like MySQL, PostgreSQL, or SQLite!
